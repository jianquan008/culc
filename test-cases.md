# 科学计算器自动化测试用例

## 功能测试用例

### TC001: 基本三角函数测试
**测试目标**: 验证sin、cos、tan函数的计算准确性
**前置条件**: 应用已启动，处于科学模式
**测试步骤**:
1. 输入30，点击sin按钮
2. 验证结果为0.5
3. 输入60，点击cos按钮  
4. 验证结果为0.5
5. 输入45，点击tan按钮
6. 验证结果为1

**预期结果**: 所有三角函数计算结果准确
**验收标准**: 计算误差 ≤ 1e-15

### TC002: 角度模式切换测试
**测试目标**: 验证DEG/RAD/GRAD模式切换功能
**前置条件**: 应用已启动，处于科学模式
**测试步骤**:
1. 确认当前为DEG模式
2. 输入π，点击sin按钮，验证结果接近0
3. 切换到RAD模式
4. 输入π，点击sin按钮，验证结果接近0
5. 切换到GRAD模式
6. 输入200，点击sin按钮，验证结果接近0

**预期结果**: 不同角度模式下计算结果正确
**验收标准**: 角度转换精度误差 ≤ 1e-15

### TC003: 对数指数函数测试
**测试目标**: 验证ln、log、exp等函数计算
**前置条件**: 应用已启动，处于科学模式
**测试步骤**:
1. 输入e，点击ln按钮，验证结果为1
2. 输入100，点击log按钮，验证结果为2
3. 输入1，点击exp按钮，验证结果为e
4. 输入2，点击10^x按钮，验证结果为100

**预期结果**: 对数指数函数计算准确
**验收标准**: 计算结果精度15位有效数字

### TC004: 科学记数法显示测试
**测试目标**: 验证大数和小数的科学记数法显示
**前置条件**: 应用已启动
**测试步骤**:
1. 计算1000000000000000（15个0）
2. 验证显示为1e+15格式
3. 计算0.000001
4. 验证显示为1e-6格式
5. 输入1.23E+5，验证解析为123000

**预期结果**: 科学记数法正确显示和解析
**验收标准**: 自动切换阈值准确，解析无误

### TC005: 内存功能测试
**测试目标**: 验证M+、M-、MR、MC内存操作
**前置条件**: 应用已启动，处于科学模式
**测试步骤**:
1. 输入100，点击M+按钮
2. 输入50，点击M+按钮
3. 点击MR按钮，验证显示150
4. 输入30，点击M-按钮
5. 点击MR按钮，验证显示120
6. 点击MC按钮，点击MR验证显示0

**预期结果**: 内存操作功能正常
**验收标准**: 内存计算准确，状态正确保存

## 边界测试用例

### TC006: 三角函数边界值测试
**测试目标**: 验证三角函数在边界值的处理
**测试步骤**:
1. 输入90度，点击tan按钮，验证显示"数学错误"
2. 输入-1，点击asin按钮，验证结果为-90度
3. 输入1.1，点击asin按钮，验证显示"定义域错误"

**预期结果**: 边界值和错误情况正确处理

### TC007: 对数函数边界值测试
**测试目标**: 验证对数函数的边界值处理
**测试步骤**:
1. 输入0，点击ln按钮，验证显示"数学错误"
2. 输入-5，点击log按钮，验证显示"数学错误"
3. 输入1，点击ln按钮，验证结果为0

**预期结果**: 负数和零的对数正确报错

### TC008: 大数计算测试
**测试目标**: 验证极大数值的计算处理
**测试步钮**:
1. 计算999^999，验证是否溢出处理
2. 计算非常小的数值运算
3. 验证精度保持和科学记数法显示

**预期结果**: 大数计算不崩溃，正确处理溢出
